***********Comparações****************
Comparação de tipo por referencia, tipos em lugares diferentes na memoria, tipos complexos são por referencia. object == object
Comparação de tipo de valor, guarda valores 1 == 1
Como os dois objetos apontam para o mesmo lugar, se um altera reflete no outro


O namespace faz parte do nome completo de uma classe.
Os nomes de namespace não devem conter traços (-) ou iniciar com números.

value só funciona dentro do set

Se expandir algum dos acessor das propriedades é obrigado a expandir os dois get e set

vantagem get e set:A implementação interna pode ser modificada sem afetar nenhum código fora da própria classe.
